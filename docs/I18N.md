# i18n (Çok dilli metinler)

Projede Türkçe (tr) ve İngilizce (en) desteklenir.

## Mobile (Expo)

- **Metin dosyaları**: `mobile/i18n/tr.json`, `mobile/i18n/en.json`
- **Kullanım**:
  - `import { getText, setLocale } from "mobile/i18n"` veya `@/i18n` (alias varsa)
  - Bileşenlerde: `import { useI18n } from "mobile/i18n/I18nProvider"` → `const { t, locale, setLocale } = useI18n();` → `t("result.openOnMap")`
- **Provider**: Uygulama köküne `<I18nProvider defaultLocale="tr">` ekleyerek locale değişince yeniden render alınır.
- **Cihaz diline göre**: `expo-localization` ile `getLocales()` alıp ilk dil koduna göre `defaultLocale` verebilirsiniz.

## Dashboard (Next.js)

- **Metin dosyaları**: `dashboard/messages/tr.json`, `dashboard/messages/en.json`
- **Kütüphane**: `dashboard/lib/i18n.ts`
  - `getText(locale, key)` – örn. `getText("tr", "nav.sessions")`
  - `getTextDefault(key, locale?)` – varsayılan locale ile
  - `parseLocaleFromHeader(header)` – Accept-Language’dan locale çıkarır
- **Sunucu bileşenlerinde**: `headers()` ile `accept-language` alıp `parseLocaleFromHeader` ile locale belirleyip `getText(locale, "nav.sessions")` kullanın.
- **İstemci bileşenlerinde**: URL parametresi veya cookie ile locale tutup `getText(locale, key)` kullanın.
